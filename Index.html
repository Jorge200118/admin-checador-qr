Bienvenido al asistente de alta de empleados

Este tutorial te guiará paso a paso para registrar un nuevo empleado en el sistema de checador QR.

¿Qué necesitas tener listo?

  • Código único del empleado
  • Nombre completo
  • Sucursal donde trabajará
  • Puesto o cargo
  • Horario asignado
  • Foto de perfil (opcional)
Tiempo estimado: 2-3 minutos

Paso 1: Datos básicos del empleado

Obligatorio

Código de Empleado

Es el identificador único del empleado. Puede ser un número de nómina, un código alfanumérico, etc.

💡 Consejos:
  • Usa un formato consistente (Ej: EMP001, EMP002)
  • No puede repetirse con otro empleado
  • Asegúrate de que sea fácil de recordar
Obligatorio

Nombre

Nombre(s) del empleado.

Obligatorio

Apellido

Apellido(s) del empleado.

Paso 2: Sucursal y puesto de trabajo

Obligatorio

Sucursal

Selecciona la sucursal donde el empleado estará asignado.

💡 Importante:
  • Esta información se usará para filtrar reportes
  • El empleado solo aparecerá en las tablets de su sucursal
  • Puedes cambiar la sucursal después editando el empleado

Puesto

Especifica el cargo o puesto del empleado.

💡 Ejemplos comunes:
  • VENTAS
  • CHOFER
  • TRABAJADOR DE PATIO
  • AUXILIAR CONTABLE
  • ENCARGADO DE SUCURSAL
  • CAJERA
  • ALMACEN

Paso 3: Asignar horario de trabajo

Obligatorio

Horario de trabajo

Selecciona el horario que seguirá el empleado. Este horario determina las horas de entrada y salida esperadas.

Importante: El horario debe existir previamente. Si no encuentras el horario adecuado, créalo primero en la sección "Horarios".
💡 ¿Qué incluye un horario?
  • Hora de entrada y salida
  • Bloques de trabajo (si hay turnos partidos)
  • Tolerancia de retardo
  • Horas objetivo diarias

Paso 4: Foto y configuración adicional

Haz clic para seleccionar una foto

Opcional

Foto de perfil

Sube una foto del empleado para facilitar su identificación.

💡 Recomendaciones:
  • Usa una foto reciente y clara
  • Formato recomendado: JPG o PNG
  • Tamaño máximo: 5 MB
  • La foto aparecerá en la tablet al hacer check-in

Trabaja domingos

Marca esta casilla si el empleado trabaja los domingos.

💡 Nota: Esto afecta cómo se calculan las asistencias y reportes semanales.

¡Listo para guardar!

Una vez que hayas llenado todos los campos obligatorios, presiona el botón "Guardar" y el empleado será dado de alta en el sistema.

¿Qué sigue después?

  1. El empleado aparecerá en la lista de empleados
  2. Podrás imprimir su código QR para que lo use en las tablets
  3. Empezará a registrar asistencias según su horario
  4. Podrás ver sus registros en la sección "Registros"
